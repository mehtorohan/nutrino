NUTRINO

"Guide to help users calculate BMI, BMR, and create a respective diet using Nutrino"

0. Add App To Home Screen:
Press the "three dot" icon in the upper right to open the menu.
Select "Add to Home screen." Press the "Add" button in the popup.

1. Introduction and Registration:
-Provide an overview of your web app's purpose.
-Implement a user registration system for account creation.

2. User Profile Creation:
-Prompt users to input personal information like age, gender, height, weight, and activity level.

3. BMI Calculation:
-Calculate BMI using the formula
-Categorize the BMI result into ranges (e.g., underweight, normal weight, overweight, obese).

4. BMR Calculation:
-Calculate Basal Metabolic Rate (BMR) based on formulas adjusted for gender, age, weight, and height.

5. Activity Factor and TDEE:
-Allow users to select their activity level.
-Multiply BMR by the activity factor to calculate Total Daily Energy Expenditure (TDEE).

6. Diet Plan Generation:
-Enable users to set goals (e.g., weight loss, maintenance, gain).
-Create a caloric deficit or surplus by adjusting TDEE.
-Suggest a macronutrient distribution and generate a sample meal plan.

7. Tracking and Monitoring:
-Allow users to track daily food intake and add custom entries.
-Display total calories consumed and compare with recommended intake.

8. Support and Feedback:
-Provide a support system for user inquiries, issue reporting, and feedback.
